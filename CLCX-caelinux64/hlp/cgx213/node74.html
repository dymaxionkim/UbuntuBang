cut 

   'cut' <pnt|nod> [<pnt|nod> <pnt|nod>]
This keyword is used to define a cutting plane through elements to visualize internal results. The plane is either defined by three nodes or points, or, in case a dataset-entity of a vector was already selected, by just one node or point. The cutting plane is then determined by the direction of the vector (displacements, worstPS). The menu option ''Show Elements With Light'' or the commands ''ucut'', ''view surf'' or''view volu'' will display the whole model again and will delete the plane. This command is intended for batch-mode. See ''qcut'' for the cursor controlled command.
