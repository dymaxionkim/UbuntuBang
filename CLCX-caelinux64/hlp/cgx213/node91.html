font 

   'font' 'd'|'l' <value(1-6)>
This command changes the drawing-font. Six fonts of different heigh are available:

font d 6

selects the highest font for the drawing area were

font l 1

selects the smallest one for the legend. The default is selected if no number is specified:

font d or font l
